  	in Main window's Debug, list down to add x64 compiler.

2.FaceDetection is a single directory that can generate dll and lib to be called.
  note:
	1)in generating dll,there is a error you may meet:_CRT_SECURE_NO_WARNINGS
   	  Set here:Project->Property->C/C++->Preprocess->Preprocess define.
	2)add a head file.the content is as follow:facedetect.h
	#include <stdlib.h> 
        #include <float.h> 
        #include <ctype.h>
        #ifdef FACEDETECT_EXPORTS
        #define FACEDETECT_API __dec1spec(dllexport)
        #else
        #define FACEDETECT_API __declspec(dllexport)
        #endif
        using namespace std;
        FACEDETECT_API int facedetection(void);
     

3.CallFaceDetection is a main directory that can call dll and lib generated by FaceDetection.
	include "facedetect.h"
	#define FACEDETECT_API __dec1spec(dllexport)
	FACEDETECT_API int facedetection(void);
